📊** Blinkit Sales Analysis Dashboard (Power BI)**

🚀 Project Overview

This project presents an interactive Power BI dashboard analyzing sales data from Blinkit – India’s Last Minute App. The dashboard provides key business insights into sales performance, outlet distribution, item categories, and customer preferences.

It is designed to help stakeholders make data-driven decisions by identifying trends, high-performing segments, and improvement areas.

📌 Key Features

📈 Total Sales Overview – Displays overall revenue ($1.2M)

📊 Average Sales – Average revenue per transaction

📦 Number of Items – Total items sold (9K)

⭐ Average Rating – Customer satisfaction score (4.0)
